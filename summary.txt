### Cell 11: Code - Aggressive Time-Based Missing Value Correction
- **Purpose**: Fills missing values using time-based methods, resamples, and saves cleaned data.
- **Methods and Functions**:
  - Sentinel handling: Similar to Cell 8, sets high values (>500 for PM, >100 for humidity, >50 for temperature) to NaN. Methodology: Detects data entry errors or sensor failures.
  - `df[col].rolling(window=10, center=True, min_periods=1).mean()`: Computes rolling mean over 10 samples (centered). Methodology: Local average imputation for time-series gaps; min_periods=1 allows partial windows.
  - `df.fillna(rolling_mean)`: Fills NaNs with rolling mean. Methodology: Time-aware interpolation using nearby values.
  - `df.fillna(method='ffill').fillna(method='bfill')`: Forward-fill then backward-fill remaining NaNs. Methodology: Propagates last known values for edge cases.
  - `df.resample('H').mean()`: Resamples to hourly frequency using mean aggregation. Methodology: Aggregates sub-hourly data to consistent hourly intervals; 'H' = hourly.
  - `df.to_csv(output_file)`: Saves the resampled DataFrame to CSV. Methodology: Exports cleaned data for Phase 3.
- **Formulas**:
  - Rolling Mean: For each position i, mean = (1/window) * Σ (x_{i - window/2} to x_{i + window/2}) where available.
  - Resample Mean: For each hour, mean = (1/n) * Σ values_in_hour.
- **Methodology**: Aggressive imputation prioritizes time-series continuity; rolling windows capture local trends; resampling standardizes frequency for modeling.
