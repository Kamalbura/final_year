\chapter{Executive Summary}
This document presents a comprehensive benchmark of three distinct machine learning architectures for 24-hour Air Quality Index (AQI) forecasting across three major Indian cities: Delhi, Hyderabad, and Bengaluru. 

The models evaluated are:
\begin{itemize}
    \item \textbf{Long Short-Term Memory (LSTM)}: A standard deep learning sequence model.
    \item \textbf{Random Forest (RF)}: A classical machine learning ensemble.
    \item \textbf{Transformer}: A modern attention-based architecture.
\end{itemize}

All models were trained using a 168-hour (7-day) lookback window to predict a 24-hour horizon. The input features include PM2.5, PM10, Carbon Monoxide, Nitrogen Dioxide, and Ozone. The target variable is the US AQI. Hyperparameter tuning was performed using Optuna.
